sound_stripper.c v1.0
by Richard Cavell
June 2025

This is a small project suitable for the TRS-80 Color Computer 1/2/3
and the Dragon Data 32/64.

It takes a given file and masks the lower 2 bits of every byte.

The input file should be 8-bit PCM monaural. The output will be
6-bit PCM monaural.
